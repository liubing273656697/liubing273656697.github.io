GitHub的基本使用

Apr 18, 2018GitHub点击

Git可以通过https方式和ssh方式连接服务器上的仓库。

两者比较：
1.https： 比较方便，但是每次fetch和push代码都需要输入账号和密码，略显麻烦
2.ssh： 传输前压缩数据，传输效率高，不需要每次提供账号密码
